# TubeMetrics 🚀

A clean, fast, and modern YouTube channel analytics dashboard built with Next.js. Paste any YouTube channel URL to instantly uncover deep analytics, engagement metrics, and video performance trends.

## ✨ Features

- **Real-Time Data:** Fetches live statistics directly from the official YouTube Data API v3.
- **Deep Insights:** Analyzes the latest 50 videos to calculate engagement rates, view velocity, and likes-to-comments ratios.
- **Interactive Charts:** Beautiful, responsive data visualization using Recharts.
- **Advanced Filtering:** Sort videos by Views, Likes, Comments, or Date, and filter by timeframes (30, 90, 180 days).
- **Export to CSV:** One-click data export for further analysis in Excel or Google Sheets.
- **Demo Mode:** Built-in rate limiting (30-second cooldown) to prevent API spamming.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Charts:** [Recharts](https://recharts.org/)
- **API:** YouTube Data API v3
